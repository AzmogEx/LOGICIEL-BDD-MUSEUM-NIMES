## Logiciel de gestion de bases de données

Développement d'un logiciel de gestion de bases de données pour le Muséum d'Histoire Naturelle de Nîmes dans le cadre d'un stage de 5 semaines.
Conception et implémentation d'une solution permettant la gestion et l'organisation des collections du musée, avec une interface optimisée pour la recherche et la mise à jour des données avec affichages pour les visiteurs du muséum.
